This is the first front-end iteration of the project PlantPi. This project will be complete with the construction of the following features:

  • The ability to water houseplant(s) automatically at intervals set by the user or manually.
  • The ability to show soil moisture status for each registered plant in real time.
  • Graphical representations of historical moisture/watering trends for each registered plant.
  • Integration with Google Maps and a Weather service that updates the user with relevant weather data.
